# Plataforma de Treinamento ADTSA
Sistema web para gerenciamento e exibição de vídeos de treinamento do grupo ADTSA.

# Acesso
IP Interno: 10.81.7.103

Localização do Servidor: Sala de TI (abaixo da mesa do SITEF) - Servidor físico

# Estrutura de Arquivos
Vídeos de Treinamento
Os arquivos de vídeo devem ser armazenados no diretório:

C:\Users\adtsa\Videos\Treinamento

# Código Fonte
Todo o código do projeto está versionado e disponível no servidor. O servidor 10.81.7.103 também hospeda outros sites, automações e dashboards do grupo ADTSA.

# Como Contribuir
1. Desenvolvimento Local
Clone o repositório

Faça as alterações necessárias

Commit e push para o branch apropriado

# Deploy
Para publicar as alterações em produção:

Acesse o servidor via MSTSC (Área de Trabalho Remota)

Realize o merge das alterações no ambiente de produção

Verifique se os novos vídeos estão no diretório correto (se aplicável)

# Adicionando Novos Vídeos
Copie o arquivo de vídeo para: C:\Users\adtsa\Videos\Treinamento

Atualize o código para referenciar o novo vídeo (se necessário)

# Observações
Este é um servidor físico local - alterações devem ser feitas com cuidado

Mantenha o versionamento em dia para facilitar o rollback se necessário

Qualquer dúvida, consulte a equipe responsável
